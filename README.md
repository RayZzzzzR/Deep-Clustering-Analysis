# Deep Clustering Analysis
**Goal : Discover the natural grouping or structure in unlabeled data without predefined categories.**

**Centroid based** : K-Means, K-means++, K-medoids, mini batch K-means

**Density based** : DBSCAN, OPTICS

**Connectivity based** :  Agglomerative (Bottom-up),
 Divisive (Top-down)

**Distribution based** : Gaussian Mixture Models (GMMs)

**Graph based** : Spectral Clustering, Markov Clustering (MCL), Girvan-Newman Algorithm
, Louvain Algorithm

**Deep-Learning based** : Neural Network Clustering, Autoencoder based Clustering, Contrastive learning based Clustering

Evaluation metrics for clustring (ft: Hungarian Algorithm !!) : These scores decide how efficient is the clustering algorithm in various aspects . . .

Clustering Accuracy

Adjusted Rand Index
